ReadMe:

******** PetList ********

PetList is a pet adoption application designed to streamline the experience of finding the perfect companion for our users.

PetList uses the Pet Finder API to seed our database of adoptable pets. Users can then search those pets by both location and pet type, i.e. dog or cat. 

The User can then select desirable pets to add to their list. This list is avaible to the user at their discreation and can be updated to suit the user's needs. 

Install Instructions: Make sure you have ruby installed. Use bundler. 
git clone the link. Run Bundle install. Run ruby bin/run.rb to access the program.
